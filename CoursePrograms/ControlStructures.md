Control structures decide the order in which instructions run in a program. There are three main types:

1. **Sequential**: Code runs one line after another, by default.
2. **Selection (Conditional)**: Code runs only if certain conditions are met. Example: `if...else` statements.
3. **Iteration (Loops)**: Code repeats multiple times. Examples: `for`, `while`, and `do...while` loops.

These structures help you control the logic and flow of your code.
